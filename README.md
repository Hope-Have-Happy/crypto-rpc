# Commands

```
// get balance of all accounts
./bin/balance --node ETHNode --chain ETH

// get balance of USDC for specific account
./bin/balance --node ETHNode --chain USDC --address 0xac2c37f15B77Ac5aC56fb8643cb02cb18F82C246
./bin/send --node ETHNode --chain USDC --amount 1111111 --address 0xac2c37f15B77Ac5aC56fb8643cb02cb18F82C246

// Get balance of bitcoin wallet
./bin/balance --node BTCNode --chain BTC
```
